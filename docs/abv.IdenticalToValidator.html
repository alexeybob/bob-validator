abv.IdenticalToValidator

Extends: abv.AbstractComparisonValidator

Validates that a value is identical to another value, defined in the options. This constraint compares using ===, so 3 and "3" are not considered equal.

var validator = new abv.IdenticalToValidator(data, {"value": "the value to compare to"});
if (false === validator.isValid()) {
     validator.messages().first();
}

Properties

Stringmessage 

This is the message that will be shown if the value is not identical. Defaults to "This value should be identical to %%compared_value_type%% %%compared_value%%." You can use the following parameters in this message:

Parameter Description
%%compared_value%% The expected value
%%compared_value_type%% TThe expected value type
%%value%% The current (invalid) value

*value 

This option is required. It defines the value to compare to. It can be a string, number or object.
